Проект придерживается осмысленных версий по итерациям прототипа.

## [0.2.0] — 2026-06-15
Разворот концепции: из витрины — в **движок задач**.
### Добавлено
- Очередь задач: ручное добавление + автоопределение домена по ключевым словам.
- Четыре AI-агента (поиск работы, фитнес, быт, проекты) генерят задачи через живой вызов модели.
- Кнопка «в календарь» у задачи: создание реального события в Google Calendar с цветом по домену.
- Фильтры по доменам, тосты-уведомления.
### Изменено
- Блок расписания стал компактным «что уже в календаре» (чтение).
### Удалено
- Недельный трекер привычек из v0.1 (заменён очередью задач).

## [0.1.0] — 2026-06-15
Первый живой прототип — пассивный рабочий стол.
### Добавлено
- Живое расписание из Google Calendar на неделю вперёд.
- Ручные блоки: недельный трекер рутин (спортзал, растяжка, еда, уборка), поиск работы, проекты с прогрессом.
- Сохранение ручных данных в localStorage.

## [0.0.1] — 2026-06-13
Старт проекта обучения циклу прототипирования.
### Добавлено
- Учебный артефакт намерения: спека-промпт «Админка с ролями» (`docs/spec-admin-panel-v1.md`).
